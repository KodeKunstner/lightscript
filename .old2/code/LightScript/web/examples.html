Executing LightScript from Java

LightScript is executed via LightScript objects, which are responsible for the compilation of the script. So a minimal Java program calling LightScript needs to create a LightScript object, and the calling the LightScript objects eval method with the actual code: 

class Main {
    public static void main(String args[]) throws LightScriptException {
        LightScript ls = new LightScript();
        ls.eval("print('Hello world');");
    }
}
Notice that the eval method can take any input stream as parameter, so 
        ls.eval("print('Hello world');");
could easily be replaced with: 
        ls.eval(new FileInputStream(new File("some/filename.js")));
to execute a given file instead.

The LightScript object also keeps track of global values, so if we want to pass values from Java to LightScript, visa versa, this can be done like: 

class Main {
    public static void main(String args[]) throws LightScriptException {
        LightScript ls = new LightScript();
        ls.set("x", new Integer(6));
        ls.eval("str = 'The answer is: ' + x * 7;");
        System.out.println(ls.get("str"));
    }
}
which would print: The answer is: 42
